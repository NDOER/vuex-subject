### vue答题小例子

### 说明

> 邓小平爷爷说过实践是检验真理的唯一标准

> 学习vuex实践的小例子

### 运行
```
git clone git@github.com:YangAlas/vuex-subject.git

cd vuex-subject

npm install

npm run dev

```
